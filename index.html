Understanding Conditional Statements

A conditional statement "If P, then Q" describes a logical relationship between two parts:

Hypothesis (P)

The "if" part - the condition that triggers the conclusion

Conclusion (Q)

The "then" part - what must follow when P is true

The statement "If P, then Q" is true ONLY when P is entirely inside Q (the first scenario). In all other arrangements, we can find counterexamples where P is true but Q is false.

Real Examples

Example 1: Numbers

Statement: "If a number is divisible by 6, then it is divisible by 3"

This is TRUE because every multiple of 6 (like 6, 12, 18) is also a multiple of 3.

Visually: The "divisible by 6" circle fits entirely inside the "divisible by 3" circle.

Example 2: Geometry

Statement: "If a shape is a square, then it has four sides"

This is TRUE because every square must have exactly four sides.

Note: The reverse isn't true - not all four-sided shapes are squares!

Finding Counterexamples

A statement is false if you can find even one case where P is true but Q is false.

Example: "If a number is prime, then it is odd"

Counterexample: 2 is prime but not odd, so this statement is FALSE.
